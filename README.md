# 🎲 D&D Мастер — ИИ-помощник для игры в Dungeons & Dragons

Веб-приложение на **Flask**, которое использует **YandexGPT** и **Yandex Art** для создания живого Dungeon Master'а. Игрок описывает действия, мастер запрашивает броски кубиков, описывает результаты и даже генерирует изображения ключевых сцен.

## ✨ Возможности

| Функция | Описание |
|---------|----------|
| 🎲 **Броски кубиков** | Автоматическая проверка успеха d20 с правилами D&D |
| 🎨 **Генерация изображений** | Создание атмосферных иллюстраций через Yandex Art |
| 📖 **Пересказ истории** | Краткое резюме всех событий одним нажатием |
| 🔥 **Атмосферный интерфейс** | Тёмное подземелье с анимированными факелами |
| 💾 **Сохранение прогресса** | История игры хранится в сессии браузера |

## 🚀 Быстрый старт

### 1. Клонирование

```bash
git clone https://github.com/ваш-username/dnd-master-bot.git
cd dnd-master-bot
2. Создание виртуального окружения
Windows:

bash
python -m venv .venv
.venv\Scripts\activate
Linux/Mac:

bash
python3 -m venv .venv
source .venv/bin/activate
3. Установка зависимостей
bash
pip install -r requirements.txt
4. Настройка API-ключей
Создайте файл .env в корне проекта:

text
YA_API_KEY=ваш_api_ключ_yandex_cloud
FOLDER_ID=ваш_folder_id
Где взять: console.yandex.cloud → создать сервисный аккаунт → выдать роли ai.languageModels.user и ai.imageGeneration.user → создать API-ключ

5. Запуск
bash
python app.py
Откройте в браузере: http://127.0.0.1:5000

🎮 Как играть
text
1. Мастер описывает начальную локацию
         ↓
2. Вы пишете действие («Осматриваюсь», «Атакую гоблина»)
         ↓
3. Если нужен бросок — мастер запрашивает его
         ↓
4. Нажимаете кнопку 🎲 БРОСИТЬ
         ↓
5. Мастер описывает результат на основе выпавшего числа
         ↓
6. Игра продолжается с шага 2
Дополнительные команды
Кнопка	Действие
📖 ПЕРЕСКАЗ	Создаёт краткое резюме всей истории
🎨 ИЗОБРАЖЕНИЕ	Генерирует иллюстрацию текущей сцены
🔄 ЗАНОВО	Начинает новую игру со случайной локации

🐛 Возможные проблемы и решения
Ошибка 403 при генерации
Причина: Сервисному аккаунту не выданы роли.

Решение: В консоли Yandex Cloud → сервисный аккаунт → добавить роли:

ai.languageModels.user

ai.imageGeneration.user

Ошибка 400 / 403 при запросе к GPT
Причина: Неверный FOLDER_ID или старый API-ключ.

Решение: Проверьте .env и создайте новый API-ключ.

Cookie слишком большие
Причина: Слишком длинная история чата.

Решение: Проект автоматически ограничивает историю 20 сообщениями.
